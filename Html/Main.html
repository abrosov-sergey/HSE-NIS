Анализ снимков легочной ткани

Работу выполнил: Абросов Сергей, БПИ202

Аннотация

Web-приложение предназначено для удобного и простого анализа снимков легочных тканей.

1. Этап анализа:
  • Выявление проблемы, решаемой приложением.
    Врачам необходимо Web-приложение для быстрого и удобного просмотра, а также простой разметки снимков легочных тканей в форматах DICOM, PNG. Пациентам необходимо Web-приложение для быстрого и удобного просмотра результатов их снимков легочных тканей в форматах DICOM, PNG.

  • Выявление целевой аудитории.
    Врачи, работающие с рентгеновскими снимками. Пациенты, получившие результат исследований.

  • Выявление вариантов использования (7±2 основных).
    Выделенные варианты использования являются вариантами использования минимально жизнеспособного продукта (MVP).
2. Этап проектирования:
  • Написание сценариев использования приложения.
  • Написание сценариев взаимодействия элементов приложения.
  • Написание сценариев выполнения скриптов.
3. Этап реализации:
  • Подготовка структуры папок.
  • Подготовка файлов.
  • Написание кода.
4. Этап проверки:
  • Проверка реализации каждого варианта использования.
  • Устранение ошибок в коде.
  • Проверка возможности использования приложения для решения выявленной на этапе анализа проблемы.
